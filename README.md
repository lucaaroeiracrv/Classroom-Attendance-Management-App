# 🏫 **Classroom Attendance Management App**

## 📋 **Descrição**

O **Classroom Attendance Management App** é um aplicativo React Native projetado para **gerenciar turmas**, **marcar presença** e **visualizar estatísticas** de alunos. Ideal para educadores que precisam de uma solução prática para controle de frequência e análise de dados.

## 🚀 **Funcionalidades**

- **Criar e Gerenciar Turmas** 🏫
- **Cadastrar e Editar Alunos** 👩‍🎓👨‍🎓
- **Marcar Presença e Ausência** ✅❌
- **Visualizar Estatísticas de Presença** 📊
- **Histórico de Presenças** 📅

## 📁 **Estrutura do Projeto**

- **ClassScreen**: Criação e gerenciamento de turmas.
- **AssuranceScreen**: Registro de presenças e ausências.
- **ManageClassScreen**: Edição e exclusão de turmas e alunos.
- **StatisticsScreen**: Cálculo e visualização das estatísticas de presença.
